# Tic Tac Toe Game

This is a **simple Tic Tac Toe game** made with **HTML, CSS, and JavaScript**.  
Players can play against each other on the same computer.

---

## Features

- Two-player Tic Tac Toe game.  
- Highlights winning combination.  
- Shows draw or winner message.  
- Clean and interactive UI.

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## Live Demo

Play the game online here:  
[Live Demo](https://arooj563.github.io/Tic-tac-toe-game/)  


---



## Usage

- Open the **Live Demo link** to play online.    
- Click on the grid to place X or O.  
- First player to align 3 marks horizontally, vertically, or diagonally wins.
